    Abstract:

    Neste trabalho foi criada uma ontologia para o ficheiro de arquivo musical com auxilio do Protégé

    Apos planeamento da ontologia foi possivel gerar um ficheiro TTL, atraves do gerador em XSLT que converte todos os campos do ficheiro arquivo musical de XML para TTL.

    Importado ficheiro da ontologia para GraphDB e conclusão das respetivas queries SPARQL.

    Deliverables:

    • Arquivo-musica xml
    • Gerador
    • Ficheiro ttl
